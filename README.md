A webgl 3d plotting library
===========================

Installation
------------

- Requirements:
   * findlib (aka ocamlfind)
   * gen_js_api

- With opam: Not yet.

- With opam (development version): Not yet.

- Manually:
```
git clone https://github.com/mlasson/webgl-plot/webgl-plot.git
cd webgl-plot
make
make install
```
and `make uninstall` to remove installed files.
